# FRS (Functional Requirements Specification) Update Instructions

Update your FRS PDF in this folder to include:

1. **ERD** – Entity-Relationship Diagram (from previous activity)
2. **UML diagrams** – From previous activity
3. **Screenshots of the Web UI:**
   - Register page
   - Login page
   - Dashboard/Profile page
   - Logout (e.g. screenshot of Dashboard with Logout button, or after clicking Logout back to Login)

**How to capture screenshots:**
- Run the web app (`npm run dev` in `/web`) and backend (Spring Boot in `/backend`)
- Open http://localhost:5173
- Take screenshots of Register, Login, Dashboard, and Logout flow
- Insert them into your FRS PDF and export/save as PDF in `/docs`

**Expected file:** Place your updated FRS PDF in this folder (e.g. `FRS.pdf` or `FRS_IT342_Lab1.pdf`).
